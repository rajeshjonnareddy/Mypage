Steps to make the website work in LINUX OS
    Step 1: Clone or download the folder to your local directory
    Step 2: Install Nodejs and npm in to your local machine
    Step 3: Now got to cloned folder and install npm and angular
             Step i:     sudo apt-get install npm
             Step ii:    sudo npm install -g @angular/cli
             Step iii:   cd /Downloads/cloned_folder
             Step iv:    npm install
    Step 5: compile and run
               ng serve
    Step 6: open browser and enter
            localhost:4200

Steps to make the website work in Windows OS
    Step 1: install Node and angular in Windows
    Step 2: set the path for nodejs using Environment Variables.
    Step 3: install angular cli in nodejs
            Step i: npm install -g @angular/cli
    Step 4: run the program in your project folder
            Step i: npm Start
    Step 5: Open webbrowser and go to http://127.0.0.1:4200
